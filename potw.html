Problem of the Week

True or False: \[ \frac{\cos(2\theta) - \cos(4\theta)}{\cos(2\theta) + \cos(4\theta)} \equiv \tan(3\theta)\tan(\theta)\]

Last Week's Answer

Answer: True.

This can be proven using the fact that \[\sin^2 (\theta) + \cos^2(\theta) = 1\] and multiplying out the brackets.
